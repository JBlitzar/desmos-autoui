## Installation

- `git clone https://github.com/JBlitzar/desmos-autoui.git && cd desmos-autoui`
- Install deps: `uv sync; uv tool run playwright install firefox`
- Run it! `uv run app.py`

Everything should open up automatically (startup sequence takes ~30s, or slower depending on your internet / computer specs). Click the links as directed in the desmos graph to control it.
